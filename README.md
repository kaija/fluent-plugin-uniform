## Installation

### RubyGems

```
$ gem install fluent-plugin-uniform
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-uniform"
```

And then execute:

```
$ bundle
```

## Configuration

You can generate configuration template:

```
$ fluent-plugin-config-format filter uniform
```
